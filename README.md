## 개발자 글쓰기의 특징

* __정확성__
  * 틀림이 없이 확실
  * 글로 쓰인 대로 하면 버그 없이 실행
* __간결성__
  * 군더더기 없이 간단 깔끔
  * 핵심만
* __가독성__
  * 쉽게 읽힘
  * 쉬운 용어, 도식
  * 문단과 문서 전체에 체계와 위계
